# APIRestify + Angular

## API

Esta API foi desenvolvida utilizando do Node + Restify, assistindo videos aulas do Fabio Vedovelli, modificando o foco  e atualizando para o proposto no teste

## Instalando as Dependências 

Utilize o comando `npm install` pra instalar todas as dependencias de cada projeto, tanto para a API quanto para o projeto Angular, execute o comando dentro de cada diretorio de projeto

## Development server

Use `npm run dev` no diretorio da API para startar a API na porta `http://localhost:3000`

Use `ng serve` no diretorio do projeto Angular para startar o servidor Angular. Navegue para `http://localhost:4200/` para ver a aplicação rodando
